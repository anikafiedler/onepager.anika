What are media queries?

Media Queries is a CSS feature that makes it possible to define different styles for different devices, screen sizes or screen resolutions. They make it possible to design a website flexibly and adaptably without having to create multiple versions of the website.

Why are media queries important?

With the increasing use of mobile devices, tablets and desktops with different screen sizes, it is crucial that websites look good and are user-friendly on all devices. Media queries make this possible by applying specific CSS rules only when certain conditions are met (e.g. screen width, height, orientation or resolution).

How do media queries work?

Media queries work with conditions that are based on properties of the device or the viewport. These conditions can include the following:

- Width and height of the viewport (e.g. for mobile devices)

- Orientation (portrait or landscape format)

- AResolution (e.g. for Retina displays)
